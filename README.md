# 🧩 Tile Match Game

A modern, responsive Tile Match web game built with a custom JavaScript framework and a React-style component architecture. Designed for performance, scalability, and immersive gameplay.

---

## 🚀 Overview

Tile Match is an interactive memory-based game where players match tiles in pairs. The game features multiple difficulty levels, real-time tracking, and smooth animations to create an engaging user experience.

Built from the ground up using a modular and maintainable architecture, this project showcases advanced frontend engineering principles without relying on external frameworks.

---

## 🛠️ Tech Stack

- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsive design  
- **JavaScript (Vanilla)** – Core logic and custom framework  

---

## 🧠 Architecture & Design

This project uses a custom-built framework inspired by modern frontend libraries:

- ⚙️ **Modular JavaScript Framework**  
  - Clean separation of concerns  
  - Reusable and scalable components  

- ⚛️ **React-style Component Architecture**  
  - Asynchronous rendering  
  - Component-based UI system  

- 🔄 **Custom Render Pipeline**  
  - Bindable system for frame-based updates  
  - Optimised rendering for performance-critical operations  

- 🧱 **Design Patterns Used**  
  - Object-Oriented Programming (OOP)  
  - MVC (Model-View-Controller)  
  - EDD (Event-Driven Design)  
  - Service-based architecture with centralised management

## 🎮 Features

### 🎚️ Difficulty Selection
- **Easy** – Smaller grid, fewer tiles  
- **Medium** – Balanced challenge  
- **Hard** – Larger grid, increased complexity  

> Higher difficulty increases the number of tiles and game complexity.

---

### 📊 Game Trackers
- ⏱️ **Timer** – Tracks how long the player takes  
- 🖱️ **Total Clicks** – Counts player interactions  
- ✅ **Matches** – Displays successful tile matches
